[맥(Mac)용 프로그램 만드는 방법]

윈도우 컴퓨터에서는 직접 맥용 실행 파일을 만들 수 없습니다.
하지만 다행히 "GitHub Actions"라는 무료 서비스를 이용하면, 윈도우에서 작업한 코드를 맥용 프로그램으로 포장해서 다운로드 받을 수 있습니다.

=== 준비물 ===
1. GitHub 계정 (없으면 가입하세요: https://github.com/)
2. Git 설치 (이미 되어 있을 수 있습니다)

=== 단계별 진행 방법 ===

1단계: 내 컴퓨터에 파일 준비 확인
- 현재 작업 폴더(c:\worknetCrawlling) 안에 ".github\workflows\build_executable.yml" 파일이 자동으로 생성되어 있습니다. 이 파일이 "맥용 변환기" 역할을 합니다.

2단계: GitHub에 코드 올리기
(이 부분은 윈도우 터미널에서 진행하세요)

1) 깃(Git) 초기화 및 파일 추가
   cmd 창을 열고, 작업 폴더로 이동한 뒤 아래 명령어를 한 줄씩 입력하세요.
   
   cd c:\worknetCrawlling
   git init
   git branch -m main
   git add .
   git commit -m "첫 번째 버전 업로드"

2) GitHub 사이트에서 저장소(Repository) 만들기
   - GitHub 로그인 -> 우측 상단 '+' 버튼 클릭 -> "New repository"
   - Repository name에 "WorknetCrawler"라고 입력
   - "Private"(비공개) 또는 "Public"(공개) 선택 후 "Create repository" 버튼 클릭

3) 내 컴퓨터와 GitHub 연결하기
   - 방금 만든 저장소 화면에 보이는 주소(https://github.com/내아이디/WorknetCrawler.git)를 복사하세요.
   - 다시 cmd 창에서 아래 명령어 입력 (주소는 본인 걸로 바꾸세요!)
   
   git remote add origin https://github.com/내아이디/WorknetCrawler.git
   git push -u origin main

3단계: 완성된 파일 다운로드 받기
1) GitHub 저장소 페이지 상단 메뉴 중 "Actions" 탭을 클릭하세요.
2) "Build Mac & Windows Executable" 이라는 작업이 돌아가고 있을 겁니다. (초록색 불이 들어올 때까지 기다리세요. 약 3~5분 소요)
3) 해당 작업을 클릭해서 들어가면, 화면 맨 아래 "Artifacts" 영역이 있습니다.
4) 여기에 "macos-executable-zip" 파일이 보일 겁니다. 이것을 다운로드 받으세요.

4단계: 맥 사용자에게 전달
- 다운로드 받은 zip 파일을 맥 사용자에게 전달하세요.
- [중요!] 맥 사용자는 "Google Chrome" 브라우저가 설치되어 있어야 실행됩니다.
- 맥에서 처음 실행하면 "확인되지 않은 개발자가 배포했기 때문에 열 수 없습니다"라는 경고가 뜰 수 있습니다.
  -> 이 경우, 아이콘 우클릭 후 "열기"를 누르고, 경고창에서 다시 "열기"를 누르면 실행됩니다.
